Descripción del Juego:
Este proyecto consiste en un juego interactivo de Pac-Man implementado con una placa Arduino UNO, una pantalla LCD 16x2, un joystick analógico, dos botones y dos buzzers para efectos de sonido y música. El objetivo es simular una versión simplificada del clásico juego Pac-Man, donde el jugador debe recolectar puntos y evitar al fantasma.
